# 📝 Zedエディタの本質とローカルAI環境構築マスターガイド

## 📌 1. Zed（ゼッド）とは：次世代AI特化型エディタの正体

Zedは、伝説的なエディタである「Atom」や「VS Code」の開発者たちが、ゼロからRust言語で書き直した、近代最高峰の超高速コードエディタである。

### 🧠 自走力を引き出す3大アドバンテージ
1. **異次元の起動・動作速度**
   * Rust製であり、MacのGPUをフルに活用して描画するため、ファイルの読み込み、スクロール、コード補完が一切の引っかかりなく「打てば響く弾丸スピード」で動作する。
2. **AIチャット（Assistant Panel）のネイティブ内蔵**
   * 外部プラグインを追加することなく、画面の横にAIとの対話スペースを常駐させられる。コード選択からの自動修正やデバッグの効率が劇的に向上する。
3. **ローカルOllamaとのシームレスな合体**
   * インターネットを遮断した「完全オフライン状態」のまま、Mac mini（M4 Pro）の内部のAI（Qwen等）をエディタの脳として直結できる。

## 📌 2. 【永久無料・完全ローカル】環境構築の全手順

Mac mini（M4 Pro / 24GB）のパワーをフルに解放し、1円も課金されることなく永久無料で使用するためのセットアップ手順。

### 🛠️ ステップ①：Zedの導入（Apple標準ルート）
ターミナルを開き、以下のコマンドで安全に強制インストールを行う。
```bash
brew install --cask zed
```
インストール完了後、開発対象のプロジェクトフォルダへ移動し、以下を叩いてZedを起動。
```bash
cd ~/Library/CloudStorage/GoogleDrive-t_kikuchi@snafty.io/マイドライブ/local-ai-project
zed .
```
*※初回起動時にプロジェクトの信頼性を確認するポップアップ（Unrecognized Project）が出たら、チェックを入れて「Trust and Continue」を選択する。*

### 🛠️ ステップ②：Ollama（ローカルAI）との接続設定
1. Zedが起動したら、キーボードの **`cmd` ＋ `,`（カンマ）** を押して設定ファイル（`settings.json`）を開く。
2. 開いた画面に以下のJSONコードを貼り付け、**`cmd` ＋ `s`** で保存する。
```json
{
  "theme": "One Dark",
  "assistant": {
    "version": "2",
    "provider": {
      "name": "ollama",
      "api_url": "http://localhost:11434"
    },
    "default_model": "qwen3.5-9b-16k:latest"
  }
}
```

### 🛠️ ステップ③：並列自走開発画面の起動
1. 設定を保存すると、エディタのテーマが自動的に黒基調の「One Dark」に切り替わる。
2. キーボードの **`cmd` ＋ `ctrl` ＋ `a`** を同時に押すか、Macの一番上のメニューバーから **「View ➔ Toggle Assistant Panel」** をクリックする。
3. 画面左側に縦長のAIチャットスペースが出現する。
4. **【スマート同期機能】**：チャット画面のメニューから直接モデルを別のもの（`qwen2.5vl:7b`など）に切り替えると、裏側の `settings.json` も自動的に最新の状態へリアルタイム更新される。

## 📌 3. 有料プラン（Zed Pro）と無料ローカル環境の決定的な違い

画面に表示される「Try Zed Pro for Free」はクラウドAIを利用するための案内であり、今回のローカル開発では契約する必要は1ミリもない。

| 比較項目 | 🔴 有料プラン（Zed Pro） | 🟢 構築した無料ローカル環境 |
| :--- | :--- | :--- |
| **月額コスト** | 約\$20（毎月の固定費） | **永久に0円**（電気代のみ） |
| **AIの演算場所** | アメリカのクラウドサーバー | **自分のMac mini（M4 Pro）**の内部 |
| **ネット接続** | **必須**（切断されると動作不能） | **不要**（Wi-Fiオフで完全遮断動作） |
| **機密セキュリティ** | コードが外部のサーバーに送信される | **完璧**。Macの外へデータが漏洩しない |
