## 📌 第3部：【実録】Googleドライブ × GitHub 連携コマンドログ
Mac miniの「マイドライブ」内に同期フォルダを作り、GitHubアカウント（TK20260401）の専用リポジトリ（local-ai-project）へ初プッシュするまでの完全な実行履歴です。
## 1. ディレクトリ作成・移動

# Googleドライブのマイドライブフォルダを自動検知して移動
cd ~/Library/CloudStorage/GoogleDrive-*
cd "マイドライブ" 2>/dev/null || cd "My Drive"
# 引き継ぎ用のプロジェクトフォルダを作成して侵入
mkdir local-ai-project
cd local-ai-project

## 2. ユーザー登録と環境のクリーンリセット

# ユーザー名の修正登録（TK03は削除、GitHubアカウント名に統一）
git config --global user.name "TK20260401"
git config --global user.email "t_kikuchi@snafty.io"
# 以前の誤った残像設定（設定URLのちぎれ）を物理リセット
rm -rf .git  # またはFinderから隠しファイル（cmd+shift+.）で直接削除

## 3. 公式推奨フローによる一発紐付けと初プッシュ成功

# 最初のファイル（README）を作成し、Gitの部屋を初期化
echo "# local-ai-project" >> README.md
git init
# 最初の歴史としてREADMEをコミット
git add README.md
git commit -m "first commit"
# タイムラインをmainに変更
git branch -M main
# 【最重要】正確な個別リポジトリのURLを登録
git remote add origin https://github.com/TK20260401/local-ai-project.git
# 登録URLの最終確認
git remote -v# 成功表示: origin  https://github.com/TK20260401/local-ai-project.git (push)
# GitHubへのアップロード（ファーストプッシュ）完了！
git push -u origin main
